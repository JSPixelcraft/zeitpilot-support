Datenspeicherung

Die App ZeitPilot – Zeiterfassung speichert alle Daten ausschließlich lokal auf dem Gerät des Nutzers.

Es werden keine Daten an externe Server übertragen.

Welche Daten verarbeitet werden

Die App kann folgende Informationen lokal speichern:

  • Mitarbeitername
  • Kontaktdaten
  • Arbeitszeiten
  • Abwesenheiten (z.B. Urlaub oder Krankheit)
  • Arbeitscodes
  • Berechnete Arbeitgeberkosten

Diese Daten werden ausschließlich zur Nutzung der Funktionen der App verwendet.

Keine Datenübertragung

Die App:ZeitPilot – Zeiterfassung

  • sendet keine Daten an externe Server
  • verwendet keine Analyse-Tools
  • nutzt kein Tracking
  • enthält keine Werbung

PDF-Export

Beim Export von Arbeitszeitberichten als PDF wird das Dokument ausschließlich lokal auf dem Gerät erstellt.

Der Nutzer entscheidet selbst, ob und wie dieses Dokument weitergegeben wird.

Datenweitergabe

Es erfolgt keine Weitergabe personenbezogener Daten an Dritte.
