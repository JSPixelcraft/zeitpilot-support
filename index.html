FAQ

Wie sichere ich meine Daten?

ZeitPilot erstellt verschlüsselte Backups. Diese können lokal oder in iCloud gespeichert werden.

Kann ich meine Daten exportieren?

Ja. ZeitPilot unterstützt CSV-Export für Microsoft Excel und andere Programme.

Werden meine Daten in einer Cloud gespeichert?

Nein. Alle Daten werden lokal auf Ihrem Gerät gespeichert. Optional können verschlüsselte Backups in iCloud gespeichert werden.

Datenschutz

ZeitPilot speichert alle Daten lokal auf dem Gerät.

Es werden keine persönlichen Daten an externe Server übertragen.

Optionale iCloud-Backups werden verschlüsselt gespeichert und können nur mit Ihrem Passwort wiederhergestellt werden.
